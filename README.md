# CLAUDE.md (中文翻译版)

本文件翻译自 GitHub 仓库 [andrej-karpathy-skills](https://github.com/multica-ai/andrej-karpathy-skills/tree/main) 中的 `CLAUDE.md`。
**原项目说明：**
> 这是 Andrej Karpathy 编写的关于如何高效使用 AI 代码助手（如 Claude）的一系列技能和最佳实践。

**翻译目的：**
- 为中文用户提供便利，使其能快速理解文件中的核心行为准则和工作流。
- 保留原文件的结构与意图，仅进行语言转换。

**内容概要：**
本指南旨在帮助减少在使用大语言模型进行编码时的常见错误，核心原则包括：
1.  **先思考再编码** - 不要假设，明确不确定性。
2.  **简洁优先** - 遵循最小化、无冗余的代码哲学。
3.  **精准修改** - 进行外科手术式的、有针对性的代码变更。
4.  **目标驱动** - 为每项任务设立可验证的成功标准。

**建议用法：**
您可以将此文件用作个人或团队在使用 AI 编码工具时的参考清单或行为守则。
